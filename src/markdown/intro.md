# 介绍

Space UI 是一款基于 **Vue 3** 和 **TypeScript** 的 UI 组件库。

这款组件库是我为了学习Vue3写的，所以未发布至npm，从初期的 UI 借鉴，到每个组件的需求与实现，包括你现在看到的这篇文章所采取的 **markdown** 书写方式呈现至此网站，对我来说都是一次新的尝试，这期间学习到了很多知识，惊叹于各个大佬的编程技巧与封装思想。

很显然这是一个简陋的 UI 库，学海无涯，我将持续在这条路上走下去，最后附上源码 [GitHub](https://github.com/QuQyouth/spaceUI)。
